# Desafio B2WAds

Realização de desafio B2Ads (https://github.com/admatic-tool/vaga-b2wads-senior).

## Requisitos:

* docker
* docker-compose

## Para executar:

* `$ docker-compose up -d`
  * Pode ser necessário utilizar `sudo`

## Para acessar API:

* `<localhost:3001>`
  * `/buy` POST
  * `/buy/{transactionId}` GET
